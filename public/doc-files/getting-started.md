# DpccGaming Docs 快速开始

欢迎来到 DpccGaming 文档中心。本文档用于帮助你快速理解项目并完成本地运行。

## 1. 环境准备

- Node.js 18+
- npm 9+
- 可选：MySQL（需要联调后端时使用）

## 2. 安装依赖

```bash
npm install
```

## 3. 本地启动

```bash
npm run dev
```

默认地址是 `http://localhost:5173`。

## 4. 关键目录

- `src/views`：页面级组件（如 Blog、Docs、Games）
- `src/components`：可复用 UI 组件
- `src/router`：路由定义
- `public`：静态资源与可直接访问的文档文件
